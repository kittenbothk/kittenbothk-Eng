# Magnetic Sensor

![](./images/magnet2.png)

This sensor can detect magnetic field, it returns a digital value,

Returns 0 when a magnetic field is detected, otherwise returns 1.

## Specifications

- Voltage: 3.3V~5V
- Type: Digital Module
- Interface: 3Pin PH2.0

## Wiring Diagram

### Robotbit Shield

Connect the sensor to Robotbit Shield's 3Pin port.

### Robotbit
